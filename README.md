### Getting Started

#### `HTTPS Clone`

```bash
git clone -b [branch-name] https://github.com/SimpleFi-NG/code-hacking-v1.git phonebook
```

#### `SSH Clone`

```bash
git clone -b [branch-name] git@github.com:SimpleFi-NG/code-hacking-v1.git phonebook
```

### Installation

#### `Yarn Install`

```bash
yarn install
```

#### `NPM Install`

```bash
npm install
```
